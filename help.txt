/help — Получить справку
/timetable — Получить расписание за все дни, начиная с текущего
/timetable today — Получить расписание на сегодня
/timetable Дата — Получить расписание на определённую дату
/book Дата Время Длительность Описание — Забронировать аудиторию для события (только для пользователей из «белого списка»)
/unbook Дата Время — Отменить бронирование для события, начинающегося в определённое время (только для пользователей из «белого списка»)
/unbook_force Дата Время — Удалить бронирование для события, начинающегося в определённое время, даже если оно уже прошло (только для администраторов)
/whitelist — Получить «белый список» (только для администраторов)
/whitelist add Имя — Добавить пользователя с никнеймом в «белый список» (только для администраторов)
/whitelist remove Имя — Убрать пользователя с никнеймом из «белого списка» (только для администраторов)
/savedata — Сохранить все данные в файл (только для администраторов)
/logmyinfo — Вспомогательная команда, записать в лог ID и никнейм пользователя
/contactlist — Получить контактную информацию
